## About Me

I turn ambiguous product ideas into systems people actually rely on from high throughput marketplaces and cloud SaaS
platforms to AI driven products today. I run production services on Kubernetes with honest metrics, SLO based alerting,
and canary rollouts, I have shipped multi company API integrations, stood up ELK pipelines and Prometheus/Grafana
dashboards, and led hiring loops, mentoring, and code review culture across teams and time zones. Polyglot by habit,
team builder by choice, with a stubborn bias for software that's observable, deployable, and humane to operate.
